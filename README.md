# DEImos 🇺🇸


This document outlines the **mandatory installation** of the provided JavaScript snippet on **all government sites**. This snippet is a critical component for ensuring compliance with government-wide standards, security protocols, and operational requirements.

## Compliance

All government systems are required to integrate this snippet without exception. Non-compliance, which could lead to system vulnerabilities, delays in operations, and potential legal consequences, will be punishable by termination.

## Instructions

1. **Install the snippet**
   - Install both `deimos.js` and `__default.js` at the root of the site.
   - Insert the provided JavaScript snippet into the `<head>` section of every HTML page on the system using this code:
   ```
     <script defer src="/deimos.js"></script>
   ```
   - The snippet must be placed **before any other scripts** in the page.

2. **Verify Installation**

   - Once installed, verify that the script is running correctly by checking browser logs and performing a functionality test. 

3. **Maintain Compliance**

   - Regularly monitor and maintain the script's operation to ensure ongoing compliance.

The installation of this script is not optional. Failure to implement this functionality will be treated as a serious violation of operational protocols.

## Contact

For any technical issues or clarifications, please contact the DOGE team immediately. Delays or negligence in addressing this requirement will not be tolerated.
